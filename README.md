# OmniFood - Responsive Food Delivery Landing Page

## Project Overview

This project is a responsive single-page landing website for a food delivery service called OmniFood. 

## Features
- Responsive Landing Page
- Hamburger Navigation
- Hero Section
- Testimonials
- Footer

## Responsive Breakpoints
- Desktop: 1024px+
- Tablet: 768px–1023px
- Mobile: below 768px

## Project Structure

index.html
style.css
script.js
images/ ---- and all the images are there it the image floder.
README.md

## How to Run

1. Download or clone the project.
2. Extract the project folder if it is compressed.
3. Open the project folder.
4. Double-click index.html
   OR
5. Open index.html using any modern web browser.

No installation or backend setup is required.
